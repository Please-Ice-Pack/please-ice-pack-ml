# please-ice-pack-ml

## 💁‍♀️ 프로젝트 개요

**Please-Ice-Pack**은 **AI**를 활용한 패킹 전후 상품 검수와 최적 포장제 추천 솔루션입니다.
패킹은 센터 상품 출고 전 마지막 작업입니다.
패킹 단계를 강화해 오피킹을 검수하고 포장 효율을 높일 수 있습니다.

## 🌱 서비스 주요기능

- 주문 상품과 실제 상품 매칭
- 상품을 매칭한 결과로 포장제 추천
- 향후 AI 모델 정확도 향상을 위한 오류 데이터 적재
